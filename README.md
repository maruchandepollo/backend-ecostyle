# EcoStyle - Backend

Este es el repositorio del **backend** del proyecto **EcoStyle**, una plataforma dedicada a la gestión integral de espacios verdes.

## Repositorio

Puedes acceder al repositorio y clonar el proyecto desde el siguiente enlace:
[https://github.com/maruchandepollo/back-ecostyle.git](https://github.com/maruchandepollo/back-ecostyle.git)

## Tecnologías utilizadas

* **Java Spring Boot** - Framework principal para la construcción del backend.
* **Maven** - Gestión de dependencias y construcción del proyecto.
* **MySQL / MongoDB** - Base de datos para el almacenamiento de información.
* **Spring Security** - Para autenticación y autorización.
* **REST API** - Endpoints para comunicación con el frontend.

## Instalación y ejecución

Sigue estos pasos para ejecutar el proyecto en tu máquina local:

```bash
# Clonar el repositorio
git clone https://github.com/maruchandepollo/back-ecostyle.git
cd back-ecostyle

# Construir el proyecto con Maven
mvn clean install

# Ejecutar la aplicación
mvn spring-boot:run
